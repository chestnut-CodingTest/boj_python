### sol
```
score = int(input())

if(score>=90):
    print("A")
elif(90>score>=80):
    print("B")
elif(80>score>=70):
    print("C")
elif(70>score>=60):
    print("D")
else:
    print("F")
```

---

### 풀이

입력값을 정수형으로 바꿔준뒤 if - elif문을 사용한다.
