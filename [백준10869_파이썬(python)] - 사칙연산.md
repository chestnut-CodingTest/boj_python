### sol
```
a, b = input().split()
a = int(a)
b = int(b)
print(a+b)
print(a-b)
print(a*b)
print(int(a/b))
print(a%b)
```


---

### 풀이

몫을 출력하기 위해서 연산한 값을 int으로 바꿔준다. 
int(a/b)
